# go-rest-api

## Environment variables

Add a config.env file in the root

```
APP_ENV=
DB_HOST=
DB_NAME=
DB_PASSWORD=
DB_PORT=
DB_USER=
HOST_PORT=
HOST_ALLOW_ORIGINS=
JWT_SECRET=
JWT_EXPIRATION=
```
